# IMU 6轴静置噪声分析

用于 6 轴 IMU 传感器静置桌面 3 分钟静态数据采集，检测加速度、角速度原始数据波动幅值，完成传感器出厂静置稳定性验收，支持数据存储、波形绘制、阈值判定验收结果。

---

## 快速开始

### 1. 安装依赖

```bash
pip install pandas numpy scipy matplotlib
```

### 2. 安装中文字体（Ubuntu / Debian 必须）

脚本会自动检测系统中的 CJK 字体并注册，无需手动配置 matplotlib。  
若尚未安装中文字体，图表标题和标签会显示为方块，请先执行：

```bash
sudo apt install fonts-wqy-zenhei
```

安装完直接运行脚本即可，**无需**清除 matplotlib 缓存或修改任何配置文件。

### 3. 运行

```bash
python imu_noise_analysis.py [LOG文件路径]
# 示例
python imu_noise_analysis.py IMU_1gb.LOG
```

### 输入文件格式

```
time(ms),ax,ay,az,gx,gy,gz
0,0.000000 0.000000 0.000000 0.000000 0.000000 0.000000
780,-0.747736 -0.391538 -9.101476 0.004909 -0.003818 0.001636
...
```

- 时间列与数据列用逗号分隔，各轴数据之间用空格分隔
- 加速度单位：m/s²；角速度单位：deg/s
- 采样率由脚本自动从时间戳中检测

## 数据截取策略

```
原始数据时间轴
|---[首部剔除15s]---[........有效区间........]---[尾部剔除15s]---|
                              ↑
                     取时间中点的180s（3分钟）
```

1. **剔除首部 15s**：排除传感器上电后的初始化漂移和稳定过渡期
2. **剔除尾部 15s**：排除录制结束前的末端干扰
3. **取时间中点的 180s**：在有效区间中以时间中点为基准，前后各取 90s，确保分析段远离边界

> 若有效数据不足 210s，脚本自动使用全部有效数据（各剔除约 5%）并给出警告。

## 分析方法与指标

### 1. 基本统计

对 6 轴（ax、ay、az、gx、gy、gz）分别计算：

| 指标 | 说明 |
|------|------|
| 均值 (Mean) | 反映传感器零偏，静置时 ax/ay ≈ 0，az ≈ ±9.807 m/s²，gx/gy/gz ≈ 0 |
| 标准差 (σ) | 反映随机噪声幅度，越小越好 |
| 最小值 / 最大值 | 反映噪声极端值 |
| 峰峰值 (P-P) | 最大值 - 最小值，反映噪声动态范围 |

---

### 2. 零偏评估 (Bias)

**加速度计零偏**：静置时 ax、ay 应为 0，az 应等于当地重力加速度（±9.807 m/s²，符号取决于安装方向）。

```
az 偏差 = az_均值 − (±9.807)
```

**陀螺仪零偏**：静置时三轴均应为 0，实测均值即为零偏。

> 传感器未完全垂直安装时，重力会分量到 ax/ay，需通过校准矩阵或水平对齐消除。

---

### 3. 噪声密度 (Noise Density)

将时域标准差转换为频域噪声密度，用于与芯片数据手册（Datasheet）对比验证：

```
噪声密度 = σ / √(fs / 2)     单位：m/s²/√Hz 或 deg/s/√Hz
```

其中 `fs / 2` 为奈奎斯特带宽。典型参考值：

| 类型 | 优秀 | 良好 | 一般 |
|------|------|------|------|
| 加速度计噪声密度 | < 200 μg/√Hz | < 500 μg/√Hz | > 500 μg/√Hz |
| 陀螺仪噪声密度 | < 0.005 °/s/√Hz | < 0.01 °/s/√Hz | > 0.01 °/s/√Hz |

---

### 4. 合矢量稳定性 (Vector Magnitude)

计算加速度合矢量幅值：

```
|acc| = √(ax² + ay² + az²)
```

静置时 `|acc|` 应恒等于 9.807 m/s²（与安装角度无关）。其标准差直接反映三轴综合噪声水平，是评估整体稳定性的核心指标。

---

### 5. 漂移分析 (Drift)

将分析段等分为前段和后段（各取 30s，不足时取 1/3），对比两段均值差异：

```
漂移 = 后段均值 − 前段均值
```

漂移过大说明传感器在测试时段内存在热漂移或机械蠕变，需要动态零偏补偿。

---

### 6. 功率谱密度 (PSD)

使用 Welch 方法估计各轴功率谱，计算：

- **DC 成分占比**：低频能量集中度，静置时应 < 5%
- **0.5~5 Hz 占比**：对应宠物/人体运动主频段，静置时应接近白噪声分布（各频段均匀）

若某频段能量异常突出，说明存在外部振动干扰或机械共振。

---

### 7. Allan 方差 (Allan Variance)

通过对不同积分时间 τ 下的 Allan 标准差分析，识别噪声类型：

| τ 趋势 | 噪声类型 | 含义 |
|--------|----------|------|
| 斜率 −0.5 | 角度随机游走 (ARW) | 宽带白噪声，越小越好 |
| 斜率 0 | 零偏不稳定性 (BI) | 慢漂移，决定长时精度下限 |
| 斜率 +0.5 | 速率随机游走 (RRW) | 低频漂移 |

本脚本报告 τ ≈ 1s 处的 Allan 标准差作为快速参考值。

---

### 8. 低通滤波效果验证

以 Butterworth 4阶低通滤波器（截止频率 5 Hz）为例，对 az 原始数据与滤波数据进行时域可视化对比，直观评估滤波前后的噪声抑制效果。

## 综合评级标准

### 加速度计

| 等级 | 平均噪声 σ | 建议 |
|------|-----------|------|
| 优秀 | < 0.098 m/s² (< 1% g) | 可直接使用 |
| 良好 | < 0.294 m/s² (< 3% g) | 可直接使用 |
| 一般 | < 0.490 m/s² (< 5% g) | 建议低通滤波 |
| 较差 | ≥ 0.490 m/s² | 必须滤波处理 |

### 陀螺仪

| 等级 | 平均噪声 σ | 建议 |
|------|-----------|------|
| 优秀 | < 0.05 deg/s | 可直接使用 |
| 良好 | < 0.10 deg/s | 可直接使用 |
| 一般 | < 0.50 deg/s | 建议低通滤波 |
| 较差 | ≥ 0.50 deg/s | 必须滤波处理 |

### 陀螺仪零偏

| 等级 | 最大零偏 | 建议 |
|------|---------|------|
| 优秀 | < 0.1 deg/s | 无需补偿 |
| 可接受 | < 1.0 deg/s | 建议静态零偏补偿 |
| 需校准 | ≥ 1.0 deg/s | 必须出厂校准 |
